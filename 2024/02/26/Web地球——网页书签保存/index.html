Web地球——网页书签仓库

Web 地球——网页书签仓库

更新时间：2024 年 02 月 26 日 21:52:37

JasonYang 的网页书签收藏仓库

1. 科研学习

论文查找

  • SEMANTIC SCHOLAR: 比较全大多数论文都可以找到

  • Paper with Code: 深度学习方向论文的分类+code+paper 适合入门查找论文

  • Tracking: 几乎是实时更新（抓取）arXiv 上面的与Tracking词条相关的论文

  • arXiv Daily: 每日推荐 arXiv 上面计算机视觉、统计学、金融学等方向的最新论文。

  • SCI-Hub: 老毛子创建的论文共享下载网站

  • readpaper: 论文阅读记录平台

知识点讲解、在线教程

  • 跟李沐学 AI : 国内 CV 研究牲的导师！
  • Hung-yi Lee - YouTube: 李宏毅老师的机器学习系列课程，其中关于 NLP 大模型系列的课程讲解的很不错。
  • Pytorch 官方的深度学习教学课程: 特别全，从反向传播，到自监督，也有 transformer 和图神经网络等等
  • 国立台湾大学在线课程: 数学、艺术、还有文学课程
  • Github Courses: 人工智能课程列表，更新快，涵盖全。
  • CS 自学网: 系统学习计算机科学的在线学习网站，主要是一些国外顶尖大学的课程，以及一些必要工具的使用
  • 特征工程学习: 一个大佬写的笔记仓库，包括特征预处理、构建、提取、选择等典型步骤
  • Pandas 中文网: Pandas 使用教程、示例
  • Numpy 中文网: Numpy 使用教程、示例
  • Matplotlib 中文网: Matplotlib 使用教程、示例

深度学习云服务器、工具、模型

  • 在线卷积池化公式计算器

  • Git 可视化学习网站

  • Pytorch Image Model: 各种 backbone 模型

  • Netron: 可视化展现 ONNX 类型网络模型的结构和 op 算子

  • ImgAug: 图像数据增强方法

  • AutoDL 算力云: GPU 租赁平台，多种 GPU 可选，按量计费，无卡模式轻松编辑代码，实例克难迁移数据和代码，毕设好帮手。

  • HOTA Metric: 多目标跟踪指标 HOTA 提出者自己写了博文来解释 HOTA 指标的含义和有效性

  • Pytorch 速查 | Anjhon’s Blog： Pytorch 基础运算速查手册

公式编辑&写作、会议发表截止时间

  • 在线 LaTeX 公式编辑器-编辑器: 目前非会员只能可视化手动敲公式了
  • Simpletex: 非常好用，多个识别客户端，无限次数公式识别，效果还很不错，强推！
  • Latex 公式手册: 公式总结
  • Create LaTeX tables online – TablesGenerator.com: LaTeX 表格创建工具
  • 妙言: 一个轻量化的 markdown 编辑器，字体、主题很好看，要不是用了 Obsidian 真就用这个写作了
  • ccf-deadlines: CCF 会议截止日期
  • 国家自然基金查询: 可以查询心仪导师或者大牛在 1997-2021 年国家批准的自然基金项目

LeetCode 学习

  • CodeTop: 按照企业考试频率整理的 LeetCode 题目

  • 代码随想录: 按照算法题目类型和难易程度分类，每道题目都有详细的讲解，以及多种语言的题解，适合初刷。

  • 编程满天星: 秋招、春招找工作岗位信息展示。

  • 推荐一个公众号：《校招 help》也是找工作的资讯汇总。

  • 其余刷题还有《剑指 OFFER》、《Leetcode 热题 100》

2. 工具类

常用

  • LKs 有趣网站分享: 各种各样杂七杂八的有趣网站推荐（闲时可以参观一波）
  • 在线工具合集: 设计配色、视频格式转换、OCR 识别、单位转换等等
  • Wallhaven: 高清电脑壁纸库
  • 在线格式转换: 任意图像、视频、文档格式转换
  • I Love PDF: 超级好用的 pdf 文件在线合并以及压缩的网站，网站内还有各种基于 PDF 的互转工具
  • TempMail: 临时邮箱用于注册一些不重要的网站
  • 中国科学技术大学测速网站
  • MarkDown emoji
  • Pake: 基于 Rust 的多端轻应用框架，里面封装了好几个国内没提供 mac 端的应用程序，微信读书、小红书和 YouTube 等等。
  • 在线文件分享 Airportal: 无须登陆注册，互联网分享文件，适合不同电脑跨系统传输（之前 Linux 版本 QQ 贼难用，现在好用了，可能用不到这个工具了）

图片处理

  • GIF 图片制作

  • 图贴士: 图片生成 GIF

  • Removebg: 背景消除或去水印

  • Upscayl - Free AI Image Upscaler: 图像 AI 无损放大，最大可到 16x，但是目前需要下载客户端才能使用

  • 佐糖 - 照片处理: 也是图像消除、图像抠图之类的集成网站

在线图表

  • 花火图表: 在线动态图表制作。

  • Processon: 在线流程图、思维导图制作。

  • draw.io: 科研绘图必备，浅色系配色很好看，同时也能进行 Latex 数学公式的渲染。

  • Desmos: 数学函数绘图、矩阵计算、图形绘图等在线的可视化数学工具。

  • Auto draw: 在线绘制简笔画，同时会有 AI 对目前的简笔画提供绘画建议

  • 矩阵计算器: 矩阵理论学习利器，在线求特征值、SVD、QR 分解之类的

RGB 颜色、编码工具

  • RGB 与 16 进制颜色编码对照表: 可以任意指定 RGB 值

  • RGB 颜色对照表: 只有对照表

  • Highlight Code: 代码高亮 用于插入 word 文档

PPT 素材 + icon 图标

  • Shields.io: GitHub 仓库 readme 图标

  • Esayicon: 简单易用

  • Iconfont 阿里巴巴矢量图标库: 图标丰富全面

  • Fotor 懒设计: 各式各样的 PPT 贴纸

  • Pexels: 免费的高清图片素材

  • 毛笔字体生成: 可以生成多种毛笔字体保存为图像

  • 字体转换: 功能同上

  • 图钉运营网: 许多新媒体运营工具（做 PPT 也可以用，比如贴图，文字处理公众号排版等工具）

  • 無料で商用可のフリーイラスト素材｜ Loose Drawing: 日系风格的设计插画，可免费商用。

PPT 模板+简历模板

  • Officeplus: 微软官方出品的免费 PPT 模版网站，可能无法商用但是个人答辩或者组会汇报够用了，现在好多稍微好看点儿的 PPT 都开始要钱了。

  • KeyNote 模板:

  • 创可贴: 登陆获取模板

  • 第一 PPT: 免费、免登陆，就是样式有点儿丑

  • Slides Go: 国外的 PPT 模板网站，免费用户每月限制下载 10 次

软件下载

  • MacWk - 精品 mac 软件下载: 好像之前的 macworks 又回来了！
  • I tell You: 正版 windows 和 office 软件下载

语言表达、文章纠错、词频统计词云生成

  • 万词王-反向词典: 查找近义词或者查找与句子表述意思相近的词语

  • 秘塔写作猫: 修改文章措辞表达，对“得、的、地”分不清真的很有用！

  • 微词云: 统计词频并可视化成词云，总结时可用

电子书籍下载

  • 码农书籍网: 计算机编程类电子书，python java，linux 基本都有
  • GitHub 上的中文编程书目索引: 也比较全
  • GitHub 上的经典编程书目索引: 经典书籍
  • Zlibrary: Kindle 电子书下载，9000w+电子图书
  • 安娜的档案: 目前 Z-library 的替代版本，目前用的最多的搜书网站。
  • Wheremylife: Kindle 的 RSS 信息快捷推送。

RSS 订阅相关

  • NetNewsWire: Free and Open Source RSS Reader for Mac and iOS。现年头资讯太多需要用一个 RSS 订阅工具进行统一查看定时推送，推荐一下 NetNewsWire，免费、多端同步，缺点是手机端无法调整阅读字体，字太小了。
  • Kill the Newsletter!: 一些博客只能支持邮箱订阅，该网站会给你生成一个邮箱链接和与邮箱对应的 RSS 订阅源，实现了资讯统一！

机场链接

  • Duyaoss: 超级多的机场介绍与测速

  • Ghelper: 目前在用，挺稳定的，也很便宜。

  • 一元机场: 便宜机场不保证稳定性

  • 用户登录 | 三分机场: 便宜机场不保证稳定性和速度

  • 爱机场: 日常分享一些机场链接

3. 人文类

优秀图书推荐

  • 豆瓣旧版图书 TOP250: 排除最近一些畅销书的冲击，回味经典

历史

  • 全历史: 各个国家各个朝代历史文化学习

新闻、播客、BGM

  • 后续: 热点事件、新闻后续报道

  • 新闻酸菜馆:

  • 今日热榜: 微博、知乎、百度、bilibili 等很多网站的热榜

  • coffee BGM: cofee 的常用 BGM 可以在放松或者学习的时候播放

  • musicforprogramming: 编程时可播放的纯音乐曲目，界面很朋克。

  • 歌曲宝: MP3 音乐下载，特别适合用于下载教研室毕设欢送和年终总结的 BGM。

影视剧观看、下载

  • 电影海报下载: 各种电影海报下载
  • 低端影视: 在线看美剧、英剧、最新的电影等等
  • 人人电影网: 经典、热门电影百度网盘保存
  • BT 之家: 经典、热门电影 BT 下载
  • Grab4k: 蓝光 4k 电影，还有 4K 演唱会实录，片库还挺全就是蓝光真的太大了一部电影十几个 G 起步

4. 其他

英语相关

  • BrainyQuote: 英文名言名句
  • Qwerty Learner: 为键盘工作者设计的单词与肌肉记忆锻炼软件，用于英文单词学习+打字
  • 在线打字游戏: 随机打出一段英语语句，挺有趣的，可以没事打着玩。

琐碎

  • 书签地球: 分享个人书签的一个网站大概率找到宝藏网址
  • Typingclub: 在线标准打字练习网站， 超好玩，网站的动态效果做得太好了。
  • 核爆范围: 查看当前原子弹或者氢弹爆炸的范围伤害
  • 生成对称炫酷的图片: 随便移动鼠标就是生成炫酷图片
  • NVIDIA 显卡算力比较查询
  • CPU 性能天梯图
  • SMS-Activate: 租用 v 电话号码 用于其他一些网站的注册等
  • FOMEPay: 注册 v 银行卡，可用于购买美区 App 或者升级 GPT
  • 崇宇信息科技 次世代合租平台: 网飞、ChatGPT、Disney 共享账号租赁网站
  • 擲骰子 🎲: 一个可以模拟掷骰子的网站。

博客友情链接

  • 潮流周刊: 前端程序员博主的周刊发现，有科技新闻、生活好物、好用工具、生活感想等等，此外博主的页面设计真的很好看，赏心悦目。
  • Tw93’blog: 博客，更多是总结、见闻感悟。
  • Sindre Sorhus: 一个轻量化 Mac OS APP 独立开发博主，轻便基本只有一个功能，其中最好用的就是“Command X”
  • Anjhon’s Blog: 但知行好事，莫要问前程
  • Martin Danelljan 主页: 一个目标跟踪大佬的已发表的论文的主页
  • coolshell: 《跟我一起写 Makefile》在线文档作者陈皓老师个人维护的网站，会发表一些技术评论
  • 阮一峰的网络日志:
  • Vaclav Kosar’s 博客主页: 写过一些 transformer 的讲解还可以
  • Mathor’blog: 他的 Transformer 讲解很不错
